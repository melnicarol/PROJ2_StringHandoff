Output exported from eclipse as project

This eclipse project was part of the final program from my Parallel and Distributed Algorithms class.
This program allows a thread to pass a string to another thread mutually exclusively making it multi-thread safe and less vulnerable to errors.
The StringHandoffImpl.java file is the only file credited to me all other files are made to test the functions implimented in that file.
